# CodeSentinel AI 🚀

CodeSentinel AI is a modern web-based developer security assistant that helps analyze GitHub repositories for potential security risks, code quality issues, and architectural weaknesses.

This project is designed to simulate real-world DevSecOps workflows by integrating automated repository analysis, authentication monitoring, and cloud-based serverless scanning functions.

---

## 🔥 Features

- Secure authentication system using Supabase
- GitHub repository import and automated analysis
- AI-assisted security scan simulation
- Serverless edge function architecture
- Modern responsive UI dashboard
- Real-time processing workflow visualization
- Developer-focused clean UX design

---

## 🧠 Architecture Overview

The application follows a modular full-stack architecture:

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend Services:** Supabase Edge Functions
- **Authentication & Database:** Supabase
- **Build Tool:** Vite
- **UI System:** shadcn component architecture

The system is designed with separation of concerns between:

- Authentication Layer  
- Repository Processing Engine  
- Security Analysis Modules  
- UI Monitoring Dashboard
